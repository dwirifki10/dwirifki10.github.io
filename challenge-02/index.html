Diskusiku

Forum diskusi untuk berbagi cerita, pengalaman dan pengetahuan. Pada sistem diskusi ini dapat menyaring kata kasar di setiap interaksi user menggunakan algoritma Levenshtein Distance, dikembangkan menggunakan Javascript, Vue Js sebagai sisi client (Front-end), Hapi Js sebagai API dalam memfilter kata kasar dan Laravel sebagai sebuah service (Back-end). Teknologi lainya yang saya gunakan adalah : GraphQL dan Hasura

PMI Bondowoso

Server Back-end dalam project ini telah dihentikan. Dibangun menggunakan Laravel sebagai Back-end dan Vue Js sebagai Front-end, digunakan untuk menyelesaikan project akhir dalam mata kuliah Workshop Web Framework dan Workshop Mobile

MidiShare

Sebuah Web Apps Knowledge Management System yang menghubungkan pengguna dan sumber pengetahuan dari dalam maupun luar organisasi. Dikembangkan menggunakan Laravel 9 dengan arsitektur monolith.

Sistem Monitoring Air Akuarium Ikan Koi

Sebuah sistem IOT untuk monitoring kualitas air akuarium ikan koi yang dikembangkan menggunakan algoritma decision tree untuk mendapatkan hasil putusan tingkat kualitas air. Dirancang menggunakan mikrokontroller ESP8266 dengan sensor suhu (dsb18b20) serta sensor Ph air.

KosKosang

Sebuah aplikasi berbasis website untuk pencarian kost (indekos) di Kabupaten Bondowoso menggunakan PHP Native dengan mengimplementasikan metode MVC (Model, View, Controller)
